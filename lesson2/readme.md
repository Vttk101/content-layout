Карточка профиля:
- должна быть шириной 300px
- с внутренними отступами 20px
- цвет фона белый
- граница светло-серого (#ddd) цвета, тонкая и сплошная

Изображение профиля:
- ширина и высота должны быть 100px
- задайте круглую форму с помощью свойства border-radius: 50%
- нижний отступ для разделения с текстом 15px
